hexo | A4支持markdown脚标便捷显示
  1. 效果演示
  2. 使用方法
  3. 其他

效果演示

markdown文件中末尾使用如下语法，定义一个脚标：

[ ^ 1]: 第一个脚标，信息显示成功 # 把[]里的空格去除，演示需要加空格

在文件内容里想要引入脚标的位置使用如下语法：

这里有一个脚标[ ^ 1 ] # 把[]里的空格去除，演示需要加空格

将呈现如下效果：

这里有一个脚标[1]，鼠标放上去会显示内容。[2]

使用方法

你的网站文件夹，举例为website。

  1. 命令行进入到webiste目录。

  2. 执行命令安装hexo插件：

npm install hexo-reference --save
  1. 执行命令查看效果
hexo clean && hexo g && hexo s

其他

经过A4使用者AllenGo912使用测试，中国大陆使用hexo-reference插件可能会有加载速度变慢的情况（该插件自带CDN的链接缘故）。

如果需要，可安装经过@AllenGo912调整CDN后的hexo-reference-new。

1.执行命令删除hexo-reference插件

npm uninstall hexo-reference

2.执行命令安装hexo-reference-new插件

npm install hexo-reference-new --save
