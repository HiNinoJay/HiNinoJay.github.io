我的快捷键逻辑-myKeyboardShortcuts

通过Arc浏览器的调教，仿佛对mac软件快捷键的使用，一下就通透了。加速了自己使用工作软件的反应。

尤其是当我归纳出一套自己喜欢的快捷键方式逻辑，有两个极大的好处：

  • 一通百通，很多软件的快捷键默认设计逻辑很相似，上手很快
  • 对于软件可能的特例快捷键，手动去更改为熟悉的逻辑，上手也很快

所以，在此记录下自己熟悉的一套快捷键逻辑，加深记忆，形成条件反射。

注意：

  • 以下字母不区分大小写
  • 加上❕提醒经常复习

全局快捷键

默认
  • 全选：command + A
  • 剪切：command + X ❕
  • 撤销：command + Z
  • 重做：command + shift + Z
  • 保存当前修改：command + S
  • 打印：command + P

撤销见名知意，很常见，大伙儿应该都会用。重做这个翻译的不太行，以至于我相信很多人像我去年以前一样，不知道怎么恢复刚才撤销的内容，只好重新打输入一遍。

其实重做就是针对撤销操作的恢复，所以我建议翻译成恢复：）

  • mac屏幕亮度-/+ ：F9/F10
  • 关闭当前窗口：command + W
  • 全屏当前窗口：fn + F ❕

个别软件不同，Arc的全屏是 control + command + F，因为该死的设置不了为fn + F

调整
  • 截图：command + control + A

我安装了一款iShot截图软件，功能强大，所以把它的截图快捷键设置为全局，把其他软件，诸如大伙截图习惯用的QQ，Wechat的截图快捷键都关闭（免得引起快捷键冲突）。甚至我把系统的截图快捷键都关闭了。

  • ocr翻译：option + S
  • 文本输入翻译： option + D

这两个全局快捷键来自于Bob这款APP，我已付费支持，十分方便快捷好用的翻译小软件。

  • 调度中心： 鼠标4键（侧边下键）
  • 桌面：鼠标5键（侧边上键）
  • 应用程序窗口：鼠标3键（滑轮键）

APP

能调整的，都特意去调整为统一快捷键。

  • 聚焦：command + space

Vim

  • 复制当前一行: yy

Typora

默认
  • 重点 ：control + ` ❕
  • 插入表格：command + option + T ❕
  • 提升/将低标题级别：command +/- ❕

  • 选中当前词：command + D ❕
  • 选中当前格式的词或者句：command + E ❕
  • 选中当前行或句：command + L

  • 侧边栏：command + shift + L
  • 侧边栏搜索文件：command + shift + F❕

  • 跳转到文首/文末：command + up/down ❕
  • 跳转到行首/行末：command + left/right❕
  • 跳转到光标位置：command + J ❕

  • 专注模式：F8
  • 检查单词拼写：command + shift + : ❕❕
  • 代码模式: command + /

Arc

调整
  • new tab: command + space
  • 刷新：F5
  • 调试：F12
  • 复制当前页面的url：command + shift + L
  • 全屏: control + command + F ❕
  • 横切页面：control + tab

Utools

调整
  • 聚焦：option + space
  • 插件ChatGPT好友：F1

Vscode

调整
  • 聚焦：command + space
