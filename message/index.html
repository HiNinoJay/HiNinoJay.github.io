主题相关问题，请先阅读帮助手册，仍无法得到解决，在Github上发起issue吧。

  • 2023.08.21如果你的A4出现之前没有的显示问题，请更新至v1.7.1版本，更新完毕按照教程修改配置文件，或者你不想更新，可手动关闭A4配置文件中的cdn，重新部署网站即可。
# 你可能会用到的 https://ninojay.top/2023/website-cdn/
## 因为是免费CDN，所以有挂的风险，一旦发现网站样式出现原来没有的问题，请尝试关闭CDN
cdn:
  enable: false # 如果你修改了 css/js ，也许需要把cdn设置为false才会生效
  choose: jsdelivr  # aliyun | unpkg| jsdelivr  # 都调整为利于国内的cdn，建议使用默认的 jsdelivr 即可
  • 2023.10.28 目前A4暂不支持分页，如果你的A4出现了分页按钮，请修改_config.yml文件中的配置，改为如下，即可关闭分页按钮。
# Pagination
## Set per_page to 0 to disable pagination
per_page: 0
pagination_dir: page
